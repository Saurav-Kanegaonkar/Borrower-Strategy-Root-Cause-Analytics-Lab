# Data Sources

This project separates public external evidence from synthetic protected portfolio data.

## Public Data

- `public_complaint_themes.csv`: summarized issue themes from a local CFPB student-loan complaint download.
- The raw CFPB CSV is ignored by git and used only as a local input to produce the summarized theme file.

## Synthetic Data

- `borrower_segments.csv`: six lending strategy cohorts with product, channel, borrower count, balance, autopay, and income-pressure assumptions.
- `daily_borrower_metrics.csv`: 120 days of modeled delinquency, application, contact, resolution, approval, and autopay metrics by cohort.
- `intervention_playbook.csv`: candidate actions with expected delinquency lift, contact reduction, effort, and owner.
- `data_quality_checks.csv`: metric confidence gates for source timing, taxonomy mapping, scenario ownership, and segment ownership.

The synthetic data is generated by `scripts/score_operating_data.py` using fixed random seeds, documented distributions, and lending-operations assumptions. It is not actual company or borrower data.
