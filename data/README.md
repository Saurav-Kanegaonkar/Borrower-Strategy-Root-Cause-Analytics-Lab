# Data Sources

Synthetic source-style data for student lending borrower strategy analytics.

- `entities.csv`: 36 borrower cohort records.
- `daily_metrics.csv`: 5,040 borrower cohort-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
